后端 Java 开发方向

ACAT 实验室后端 Java 方向主要学习内容包括：

  • 精通 Java 核心语法，深刻理解面向对象编程（OOP）思想
  • 熟练运用 SSM、Spring Boot 等主流开发框架
  • 掌握 MyBatis 数据库操作，熟练进行 CRUD 与复杂查询
  • 熟练使用 Maven 进行项目构建、依赖管理、打包部署，理解依赖传递、生命周期等核心概念
  • 熟练使用 Redis（缓存、分布式锁等场景）、RabbitMQ / Kafka 等中间件，理解其核心原理
  • 掌握 JWT 鉴权、Redis 缓存、消息队列等企业级开发关键技术
  • 掌握 Git 版本控制及基本 Linux 操作命令，支撑日常开发与部署工作

适合希望进入企业级后端开发领域的同学。
