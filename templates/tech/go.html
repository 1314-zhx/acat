后端 Go 开发方向

ACAT 实验室后端 Go 方向主要学习内容包括：

  • Go语言基础与并发编程
  • Go函数式编程设计，优雅的Go语言风格
  • net/http Go语言原生网络编程包学习
  • 数据库操作 MySQL学习，Redis学习，Gorm学习
  • zap,sqlx等 第三方库学习
  • 网络协议学习:(http/https tcp/udp)
  • web框架开发(Gin)
  • RESTful API 设计与最佳实践
  • Gin/Gorm web实战，动手写一个自己的网站
  • 分布式与微服务
  • 容器化部署：docker与kubernetes
  • 更多组件学习
  • 拓展学习：消息队列、缓存策略、监控告警等组件

适合希望进入企业级后端开发领域的同学。
